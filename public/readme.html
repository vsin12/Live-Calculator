Task details :

  1. Developed this app using Node.js, socket.io, JQuery, HTML5, CSS (Bootstrap)
  2. Did not use any database, instead I am storing everything in localstorage (to maintain data in-between sessions)
  3. Operators available :
    1. For addition : +
    2. For substraction : -
    3. For multiplication : *, x, X
    4. For division : /
    5. For Modulo : %
    6. For Power : ^,**
  4. Check for invalid expressions (multiple operators, NaN values, misplaced operators) is being done
  5. These operators will be replaced with whitespace for evaluation : {, }, (, ), _
  6. Note : a++b works as a + (+b)

Future scope :

  1. Better solution can be done for localstorage (because end-user can just go to developer tools and clear it )
  2. BODMAS can be implemented with some more time
  3. Note : I tried implementing global user count but there seems to be some issue with socket.io's 'connect' - it treats every refresh command as a new connection
